#  Intégration de Données Ontario 511 via SQL Server

##  Description
Ce projet vise à **collecter, intégrer et analyser les données de circulation routière Ontario 511** à l’aide de **SQL Server**.  
Il permet de centraliser les informations liées au trafic (incidents, conditions routières, fermetures, travaux, etc.) dans une base de données relationnelle afin de faciliter leur **exploitation analytique et décisionnelle**.

---

##  Objectifs
- Intégrer les données Ontario 511 dans une base **SQL Server**
- Concevoir un **modèle de données relationnel optimisé**
- Nettoyer, transformer et structurer les données
- Réaliser des **requêtes SQL analytiques**
- Produire des vues et rapports exploitables

---

##  Technologies utilisées
- **SQL Server**
- **T-SQL**
- **SQL Server Management Studio (SSMS)**
- **Fichiers CSV / API Ontario 511**
- Scripts ETL (Extract, Transform, Load)
